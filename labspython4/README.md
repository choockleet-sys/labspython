# Использованные источники:
1. [#44. Замыкания в Python](https://proproprogs.ru/python_base/python3-zamykaniya-v-python)

2. [Декораторы Python](https://pythonworld.ru/osnovy/dekoratory.html)

3. Замыкание — это внутренняя функция, которая запоминает переменные из внешней функции и продолжает иметь к ним доступ даже после того, как внешняя функция завершила работу.

4. Декоратор — это функция, которая принимает другую функцию, оборачивает её дополнительным поведением и возвращает новую функцию. Исходная функция при этом не изменяется.
